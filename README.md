<h2>Description</h2>
Deployed an Azure VM as a publicly accessible honeypot to attract real world attacks. 
Forwarded logs to a centralized repository and integrated Microsoft Sentinel to monitor and analyze malicious activity.
Gained hands on experience in cloud security, log management, and SIEM based threat detection.
<br />


<h2>Tools & Technologies</h2>

- <b>Microsoft Azure (Virtual Machines, Network Security Groups)</b> 
- <b>Microsoft Sentinel (SIEM)</b> 
- <b> Log Analytics Workspace</b> 


<h2>Environments Used </h2>

 <b>Windows 10 & 11 </b>  

 <h2>Key Features</h2>

- <b>Created a virtual machine on Azure and configured it with a public IP.</b>
- <b> Opened necessary ports (e.g., RDP/SSH) to attract attackers.</b>
- <b> Forwarded logs from the VM to a Log Analytics Workspace.</b>
- <b>Integrated Microsoft Sentinel for real-time monitoring and analysis.</b>
- <b> Observed and analyzed real-world attack patterns</b>

<h2>Skills Demostrated:</h2>

- <b>Cloud Infrastructure Deployment (Azure)</b>
- <b> Honeypot Configuration & Exposure</b>
- <b>Log Forwarding & SIEM Integration</b>
- <b> Threat Detection & Security Monitoring</b>
- <b> Basic KQL (Kusto Query Language) for log analysis</b>
